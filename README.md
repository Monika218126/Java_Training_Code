# Java_Training_Code

# 🚀 Java Logic Building Practice

This repository contains a collection of **logic-building problems implemented in Java**.
The goal of this project is to strengthen **problem-solving skills**, improve **core programming logic**, and prepare for **coding interviews**.


---


📌 Topics Covered

🔥 Section 1 — Number Logic

* Reverse Integer with Overflow Handling
* Armstrong Number (Dynamic Digits)
* Strong Number

⚡ Section 2 — Loops & Conditions

* Prime Numbers in Range (with count)
* GCD and LCM (without built-in functions)
* Perfect Number Check

🔁 Section 3 — Arrays (Logic Heavy)

* Move All Zeros to End
* Find Duplicate Element (without extra space)
* Leaders in Array
* Left Rotate Array by K Steps
* Subarray with Given Sum

 🔤 Section 4 — Strings

* Longest Substring Without Repeating Characters
* String Compression

 🔁 Section 5 — Patterns

* Hollow Pyramid Pattern

 💣 Section 6 — Advanced Logic

* Frequency of Elements in Array (without HashMap)
